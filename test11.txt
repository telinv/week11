1. Что такое DOM?
-Document Object Model- интерфейс, позволяющий JS работать с разметкой документа; независимый от платформы 
и языка программирования формат, который позволяет программам получать доступ к содержимому HTML-документов 
и изменять их структуру, содержимое и дизайн.  

2. Что такое BOM?
- Browser Object Model- набор глобальных объектов, которые управляют поведением браузера,предоставляет доступ 
к navigator, screen, location, frames,history

3. Каким методом можно получить элемент по его идентификатору в DOM?
- с помощью getElementById/getElementsByClassName/getElementsByTagName/querySelector и querySelectorAll

4. Каким методом можно создать новый элемент в DOM?
-с помощью createElement или методом append

5. Каким методом можно добавить класс к элементу в DOM?
-с помощью метода toggle или add 

6. Каким методом можно изменить текстовое содержимое элемента в DOM?
-с помощью cвойствa textContent

7. Что такое объект события и какие у него могут быть свойства?
-событие-то, что происходит когда мы делаем что-то на странице, объект содержит информацию о произошедшем событии.
свойства:bubbles,cancelable, currentTarget,defaultPrevented,eventPhase,isTrusted
true/false,target,timeStamp,type

9. Для чего используется запись: `setAttribute(name, value)`?
-Метод setAttribute принимает на вход два аргумента: имя атрибута, значение которого мы хотим 
задать, и само значение.Оба аргумента — строки. Если передать значение другого типа, оно всё равно 
будет приведено к строке. Она  устанавливает значение атрибута.
